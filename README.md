# blog-images_sw

The Celeb (https://thecelebissue.blogspot.com/) 블로그용 공개 이미지 CDN 저장소.

- 폴더 형식: `YYYY-MM-###-English-Topic-Slug/` (blog-content_sw와 번호·날짜 동일, 주제만 영문 SEO)
- 파일: WebP (quality=88), 파일명은 영문 소문자-하이픈. 썸네일은 `*-cover.webp`
- 참조 URL: `https://cdn.jsdelivr.net/gh/acesstory-kr/blog-images_sw@main/[폴더]/[파일명]`
